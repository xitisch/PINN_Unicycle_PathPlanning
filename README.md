# PINNs_for_Path_Planning
Bachelor Thesis in Optimization
Using Physics-Informed Neural Networks to perform optimization of path planning of unicycle robot. 
The method uses boundary conditions as hard constraints. 
Besides, 3 aspects are considered when optimizing and designing the loss function, including:
1. Physics information.
2. Obstacle avoidance.
3. Optimization of path length.

Furthermore, the project will investigate how the curvature of the generated trajectory varies with obstacle radius and characteristic length. In addition, the interaction between the different loss components will be analyzed through a variational perspective, examining how the weighted residual formulation influences the optimal trajectory under the unicycle dynamics.
